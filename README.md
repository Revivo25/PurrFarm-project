# PurrFarm-project
PurrFarm is a cloud-based smart dashboard designed for precision agriculture. 
The system collects real-time IoT sensor data (temperature, humidity, soil moisture, light), allows users to upload plant images, and analyzes plant health using AI to detect diseases, pests, and water stress.
